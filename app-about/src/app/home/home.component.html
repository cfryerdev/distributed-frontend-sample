About

This is a functional example of a distributed micro-frontends architecture hosted in AWS Cloud. The application is broken up into 4 parts written in the latest in modern web technologies: Angular 9, React 16.13, Vanill JS, and Vue 2.6.

  • Home: React
  • Login: VueJS
  • About: Angular
  • Products: REact

While navigating the application, you should not be able to notice the transition between applications or technologies.

This website is made up of four distinct applications, Home, Login, About, and Products. State is managed at the cookie level to save cost, but in a production environment should be held server-side using a restful api like expressjs and a cache service such as Redis. When navigating between applications the cookie retains the data, so that each application is able to pickup where the last left off. This also allows users to freely navigate the entire sitemap without interuption.

The application also leaverages several npm packages to ensure each application has the exact same experience and functionality.

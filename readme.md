## Deploying from local

* Install AWS CLI
* Install Terraform
* Install Node / NPM
* Log into aws cli by running: `aws configure` -> [Documentation](https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-configure.html)
* Clone this directory and navigate to it
